# Requirements

This site runs on Python version 3.

[Install Python3](https://www.python.org/downloads/)

# Build and Run

``` bash
git clone https://github.com/chrisdias/thecatsaidno.git
cd thecatsaidno
pip3 install -r ./requirements.txt
flask run
```

Browse to http://localhost:5000
